How to convert IP addresses in decimal form to its binary representation?

Each octet of the IP address is converted into binary form. For example, the IP address 192.168.0.1 is converted into 11000000.10101000.00000000.00000001. If we look at the second octet as an example, each bit is assigned a value based on its position. If its place value is 128 and the bit is 1, then 128 is added to the final result. When all bits are processed, we get the decimal form of the octet.

Binary value 1 0 1 0 1 0 0 0
Place value 128 64 32 16 8 4 2 1
Result 128 0 32 0 8 0 0 0 =168
